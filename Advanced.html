Data Analyst Roadmap for Advanced Learners

Following is the roadmap to learn Data Analyst skills for a propsects with the idea of the basics of data analysis especialy excel and vissualization. It includes FREE learning resources for technical skills (or tool skills) and soft (or core) skills. .

We do more of Exploratory Data analysis to give a better understanding of datasets and how to source, clean, house and prepare data for analysis. We handle larger data, so we progress to use more of SQL.
